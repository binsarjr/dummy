# Dummy

API untuk dummy data. Mirip seperti jsonplaceholder.typicode.com

Hanya saja ini dikhususkan untuk membuat dummy data untuk kelas di Neuversity.


## Installation

```bash
$ bun install
```

## Running the app

```bash
# development
$ bun run start

# watch mode
$ bun run start:dev

# production mode
$ bun run start:prod
```
